Next steps to complete all identity verification requirements

Depending on your role, you must provide your personal code when you submit certain filings to Companies House. This is what connects your verified identity to our records. It's part of the legal requirement to verify your identity for Companies House.

This applies if you are a:

  • director (or equivalent), or are going to be one
  • person with significant control (PSC), or are going to be one

You need to use your code for each role you hold, and for every company you're involved with.

For more information, read the 'How to use your personal code' section of the Companies House personal codes for identity verification guidance (opens in new tab).

If you're currently a director or PSC

You'll need to provide your code when you submit certain filings. For example, directors will need to provide it on their company's confirmation statement. PSCs will need to provide it on their verification statement.

If you're going to be appointed as director or become a PSC

You'll need to provide the code when you submit the filing to:

  • appoint a director
  • give notice of a PSC

If you're both a director and a PSC, you'll need to provide the same personal code separately for each role, and for every company you're involved with.

If you're incorporating a company

You'll need to provide the code when you submit the filing to incorporate the company.

About your personal code

You'll only need one personal code for Companies House. You'll use the same code for all your roles and companies.

The code is personal to you, not your company or a company you work for.

You should keep your code safe. If you use an Authorised Corporate Service Provider (ACSP), such as an agent or an accountant, you will need to share it with them if they file on your behalf.

Your code is not the same as the company authentication code (opens in new tab). If you have an authentication code, you should continue to keep this safe as you'll need it to file online.

What to do if the email doesn't arrive

You can also find your personal code in the Manage account (opens in new tab) section when you sign in to your Companies House account.

If the email doesn't arrive after a few minutes, check your spam folder. If it still hasn't arrived, email us at enquiries@companieshouse.gov.uk and we'll resend it.
