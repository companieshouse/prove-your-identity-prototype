What a Companies House personal code is

You need to use your Companies House personal code to connect your verified identity to our records. This is so that we know the correct person is linked to the name we have on file. The code is personal to you, not your company or a company you work for.

You'll need your personal code for various reasons. For example:

  • as part of some other company filings, such as the confirmation statement
  • if you are appointed as a director
  • if you become a person with significant control (PSC)
